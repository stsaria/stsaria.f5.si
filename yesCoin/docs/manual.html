YesCoin

はじめに

YesCoinはYINSTのシンプルなオルトコインです。しかし、ビットコインのようなP2Pや、nonceなどの仕組みにはなっていません。
別にそんなにめっちゃ頑張って作ったわけじゃなくて、勉強用で作ったので、まあクオリティーは気にしないでもらって.....

人のノードを使ってみよう

まあこれはYINSTでしかURLを共有してないですが....
なので多分ここに書くことはないです。

ダウンロード&インストール方法

※Python3,Pip3,Gitが必要です。

    共通

  1. ダウンロード
    GithubからYesCoinをダウンロード(またはここから) 
    ~$ git clone https://github.com/stsaria/yesCoin
  2. 必要なライブラリをインストール
    ~yesCoin$ pip3 install -r requirements.txt

    ノード（マイニングとか送金とかできるクライアント）場合

  1. 実行
    ~yesCoin$ python3 src/main.py
# 初期中央サーバーが求められるので、教えられた中央サーバーアドレスを入力
初期中央サーバー(例: http://xxx.com:11380):
    操作画面はブラウザ上で行います。 実行後http://127.0.0.1:11381にアクセスすれば、ログイン画面->ダッシュボードが使えます。 もし共有するのなら、ポート11381の解放と、次の ユーザー制限 を見てください。
  2. ユーザー制限

    もし、そのノードの操作画面をいろんな人に共有するなら、マイニングとかが増えると、CPU使用率がすごく上がってしまって、サーバーが重くなってしまいます。
    それを制限したいのなら、

    DONTMINING
DONTMANUALSYNC
    のような、ファイルをyesCoinディレクトリ直下に置くと制限できます

    中央サーバーの場合

  1. 実行
    ~yesCoin$ python3 src/main.py centralServer
# 初期中央サーバーが求められるので、教えられた中央サーバーアドレスを入力
初期中央サーバー(例: http://xxx.com:11380):
    操作画面はブラウザ上で行います。 実行後http://127.0.0.1:11381にアクセスすれば、ログイン画面->ダッシュボードが使えます
  2. あと必要なこと

    1. ポート開放
    2. Systemdに登録

    /etc/systemd/system/yesCoin.service
[Unit]
Description=Virtual Coin - Yes Coin -
After=network.target

[Service]
WorkingDirectory=/home/stsaria/yesCoin

User=stsaria
Group=stsaria

Restart=always

ExecStart=/usr/bin/python3 src/main.py centralServer

[Install]
WantedBy=multi-user.target
